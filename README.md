# image2gesture
A multi-stage classical image processing pipeline for real-time hand gesture recognition. 

*For I2200/EE4710: Introduction to Digital Image Processing at City College of New York*

# Functionality
- Processes image/video to detect gestures in real time via classical image processing techniques
- Live demo output, includes filter control
- Can be evaluated against HoloLens 2 native hand tracking
